Folium Map Analysis Project

Below Tableau Dashboards were created with tables generated from SQL queries
using NYC and Chicago data from the Folium Map Project.

About Project

    Tools (Python, Microsoft SQL Server Management Studio, Tableau)
    Reduced Datasets of both the New York and Chicago Datasets
    from the Folium Map Project were used to visualize the data
    (10000 rows for each).
    Context: New York Dataset - 376640 rows, Chicago Dataset - 200088 rows
    Uploaded data into Microsoft SQL Server Management Studio to query.
    Used SQL queries to create tables to upload into Tableau Public to visualize.

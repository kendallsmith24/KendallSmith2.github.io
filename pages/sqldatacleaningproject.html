Below Tableau Dashboard was created with tables generated from SQL queries

About Project

    Tools (Microsoft SQL Server Management Studio, Tableau)
    Uploaded Nashville Housing data into Microsoft SQL
    Server Management Studio to clean the dataset.
    (e.g. Standardizing Date Format, Creating New Columns, etc.).
    Queried the dataset for various information (e.g. Avg. Land Values, Total Number of Properties).
    Created tables from SQL queries to upload into Tableau Public to visualize.

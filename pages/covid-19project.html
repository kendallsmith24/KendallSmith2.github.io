Covid-19 Project

Below Tableau Dashboard was created with tables generated from SQL queries
using Covid-19 data found on the Our World in Data* website.

About Project

    Tools (Excel, Microsoft SQL Server Management Studio, Tableau, Python)
    Extracted Dataset from Our World in Data* website.
    Preprocessed the data in Microsoft Excel.
    Uploaded data into Microsoft SQL Server Management Studio to query for
    various information (e.g. Deaths, Cases, etc).
    Used SQL queries to produce tables to upload into Tableau Public to visualize.
    Reproduced Project in Python (using Plotly for visualizations)
    * Original Source and Raw Data found at JHU CSSE COVID-19 Data
